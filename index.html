about

The engineer behind the endpoint

Three years in, I've learned that the interesting problems live on the backend — in the aggregation that won't scale, the integration that times out, the workflow that needs to survive a partner API going down.

At Stucred I build the Node.js services behind a lending product: complex MongoDB pipelines, Redis-backed performance work, AWS Lambda and Step Functions, and the verification rails that move a loan from application to approval.

I care about clean code, fast queries, and integrations that just work — and I'll happily jump to React and Redux when the frontend needs the same rigor.
